ü§ñ Zaakiy Chatbot Widget

This is a simple example showing how to embed the Zaakiy chatbot on any website. The chatbot should appear in the bottom-right corner.

üìå How it works:

  • The widget loads asynchronously without affecting your page speed
  • It automatically fetches your chatbot configuration
  • All conversations are tracked and analyzed in your dashboard
  • Works on any website - WordPress, Shopify, custom HTML, etc.

üíª Embed Code

Copy and paste this code before the closing </body> tag of your website:

<script>
  (function() {
    var script = document.createElement('script');
    script.src = 'https://zaakiy.vercel.app/chat-widget.js';
    script.setAttribute('data-chatbot-id', 'YOUR_CHATBOT_ID_HERE');
    script.setAttribute('data-api-url', 'https://zaakiy-production.up.railway.app');
    document.head.appendChild(script);
  })();
</script>

‚öôÔ∏è Configuration Options

You can customize the widget by adding these optional data attributes:

  • data-chatbot-id - Your specific chatbot ID (required)
  • data-api-url - API endpoint (default: https://zaakiy-production.up.railway.app)
  • data-position - Widget position: "bottom-right" or "bottom-left" (default: bottom-right)
  • data-primary-color - Custom color (e.g., "#ff6b35")
  • data-bot-name - Custom bot name
  • data-greeting - Custom greeting message
