🧪 Popup & Chat Interaction Test

✅ Expected Behavior:

When popup is visible and user clicks chat button:

  • Welcome popup should close smoothly
  • Chat window should open
  • No overlap between popup and chat window
  • Input field should get focus

📋 Test Scenarios:

Scenario 1: Direct Chat Button Click

1. Wait for popup to appear
2. Click the chat button (bottom-right)
3. Popup should close and chat should open

Scenario 2: "Start Chat" Button

1. Wait for popup to appear
2. Click "Start Chat" in the popup
3. Popup should close and chat should open with focus

Scenario 3: "Maybe Later" then Chat Button

1. Wait for popup to appear
2. Click "Maybe Later"
3. Click chat button
4. Only chat should open (no popup interference)
