# Zentra Frontend

This is the frontend for the Zentra project, built using [Next.js](https://nextjs.org), TypeScript, and Tailwind CSS. It provides a modern, responsive interface for managing AI-powered chatbots and dashboards.

## Features

- User authentication with Supabase
- Dashboard for managing chatbots
- Customizable chatbot configurations
- Performance monitoring and analytics
- Accessibility-first design with Radix UI components

## Getting Started

### Prerequisites

Ensure you have the following installed:

- Node.js (v16 or higher)
- npm, yarn, pnpm, or bun

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-repo/zentria-frontend.git
   ```

2. Navigate to the project directory:

   ```bash
   cd zentria-frontend
   ```

3. Install dependencies:

   ```bash
   npm install
   # or
   yarn install
   ```

4. Set up environment variables:
   - Create a `.env.local` file in the root directory.
   - Add the required environment variables (e.g., `NEXT_PUBLIC_API_URL`, `SUPABASE_URL`, `SUPABASE_KEY`).

### Running the Development Server

Start the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.
